Python CLI's Docs
======================================

Python CLI is a boilerplate project that will give your new project a great headstart!
It is built on top on amazing Open Source, community-maintained libraries and packages.

Simply clone it using git and modify it as you need:
    ``git clone https://gitlab.com/megabyte-labs/templates/python-cli.git``

Features
----------------

- `Poetry <https://python-poetry.org/>`_ as a dependency manager.
- `Sphinx <https://www.sphinx-doc.org/en/master/>`_ and Themes documentation.
- `Rich <https://github.com/willmcgugan/rich/>`_ for logging and text formatting in the terminal.
- `Click <https://click.palletsprojects.com/en/8.0.x/>`_ to create command line interfaces.
- `Fire <https://github.com/google/python-fire/>`_ to generate CLIs from Python objects.

Installation and Usage
----------------------
Make sure you install `Poetry <https://python-poetry.org/>`_ in your system. Then, install dependencies with:
  ``poetry install``

Run with:
  ``poetry run python python_cli/cli.py``
